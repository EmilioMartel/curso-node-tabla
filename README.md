# Notas:
Tabla de multiplicar por cli en Node

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     base de la tabla de multiplicar            [número] [requerido]
  -h, --hasta    límite de la tabla de multiplicar        [número] [defecto: 10]
  -l, --listar   muestra la tabla de multiplicar     [booleano] [defecto: false]
```
